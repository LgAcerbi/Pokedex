# Pokédex
## This project is currently in development

This is a pokédex that contains all pokémons contained in POKEAPI.CO
developed with this frameworks:

* Axios  
* React  
* Cors  
* Dotenv  
* Express  
* Pokemon-promise-v2 (aka POKEAPI.CO)
